# Agent 职责与边界

## 核心原则

LingoForge 的学习 Agent 是自适应学习调度员和私人辅导老师。

它不是：

- 单纯的动态出题器；
- 可随意修改成绩的评分器；
- 替代确定性程序的数据写入者；
- 多个独立 Agent 的集合；
- 只靠泛化提示词生成“像六级”的内容生成器。

主线原则为：

**真题为源，Skill 为教学能力，Agent 负责调度，确定性程序负责判分，未见真题负责验收。**

## 四类职责边界

### 产品需求

产品需求定义用户要完成什么学习闭环：

- CET-6 词汇与阅读；
- 首次目标收集；
- 短诊断；
- 基于 4 类能力维度的最小画像；
- 真题炼化 Skill 驱动训练；
- 第一次完整主线学习；
- 一个机场购票副线任务；
- 副线信号回流主线候选池或待验证标记；
- 第二次自适应主线计划；
- 至少一个体现自适应变化的短训练任务；
- 短隔离检测。

### Agent 自主决策

Agent 在程序约束内负责：

- 理解用户目标；
- 读取用户画像和历史表现；
- 决定当前最值得训练的能力；
- 在程序候选词范围内选择今日目标词；
- 决定新词与复习词比例；
- 决定材料主题、难度、长度和提示等级；
- 选择合适的 CET-6 阅读 Skill；
- 根据用户中途表现修改训练计划；
- 分析错误原因；
- 决定补讲、降难度、保持或升难度；
- 提出结构化画像更新建议；
- 将部分薄弱词映射到副线场景曝光。

Agent 必须给出可追溯依据，不能只给出“根据你的情况”这类泛化解释。

### 固定 Workflow

固定 Workflow 规定流程顺序：

- 首次目标收集；
- 短诊断；
- 初始画像；
- 候选词筛选；
- Agent 调度；
- 低压力学习；
- 真题风格迁移训练；
- 程序判分；
- Agent 错因分析；
- 程序校验画像更新；
- 机场副线任务；
- 副线信号回流；
- 第二次主线自适应计划；
- 第二次短训练任务；
- 短隔离检测。

Agent 可以在 Workflow 内调度，但不能跳过必须的判分、校验、隔离和记录环节。

### 确定性程序

确定性程序负责：

- 计时；
- 客观题判分；
- 测试题隔离；
- 防止重复抽题；
- 原始行为记录；
- 数据合法性校验；
- 数据库存取；
- 候选词预筛选；
- 核心成绩和正式画像更新校验；
- 题目版本、Skill 版本和生成版本记录；
- 页面交互和角色动画；
- 副线任务客观结果记录。

确定性程序决定是否应用 Agent 的画像更新建议。

## Agent 不得越界

Agent 不得：

- 直接修改核心成绩；
- 直接修改正式能力画像；
- 直接把副线表现认定为 CET-6 能力提升；
- 绕过候选词筛选自由选择目标词；
- 让训练内容访问隔离题集；
- 根据本轮用户表现临时生成隔离题后自行验证；
- 把模型自称“质量检查通过”当作质量校验结果；
- 宣称已经证明长期提分或训练具有因果效果。

## 用户画像更新职责

MVP 采用“4 类 Skill 对齐 + 提示依赖横向记录”的最小画像模型。

4 类能力维度：

1. 词汇语境识别；
2. 长难句与逻辑关系；
3. 同义替换与定位；
4. 干扰项判断。

横向记录尽量与具体能力维度关联：

- 提示依赖程度；
- 常见错误类型；
- 最近证据来源；
- 当前置信度。

画像更新流程：

1. Skill 声明可观察能力和证据要求；
2. 用户完成任务；
3. 程序判分并记录行为；
4. Agent 根据实际证据提出结构化画像更新建议；
5. 程序校验后决定是否应用。

Agent 的画像更新建议必须包含：

- 能力维度；
- 变化方向；
- 理由；
- 证据引用。

当前规格阶段只确定画像语义和更新原则，不确定数据库结构、数值评分公式、权重、阈值或单次更新幅度。

## Skill 的定位

Skill 是结构化教学能力，不是独立 Agent。

Skill 来自 CET-6 真题或合法考试样题中提炼出的规律，包括：

- 能力标签；
- 题型规律；
- 语言难度；
- 同义替换方式；
- 干扰项模式；
- 评价标准；
- 可观察证据要求；
- 质量校验要求。

同一个学习 Agent 根据用户画像和本轮表现调度不同 Skill。

## Skill 统一契约语义

MVP 采用“统一公共契约 + 按任务类型区分的载荷”。

当前规格阶段只确定统一契约的语义边界，不锁定最终 JSON Schema、字段类型、接口设计或具体实现。

### 公共输入语义

公共输入至少包括：

- Skill 标识与版本；
- 目标能力；
- 当前任务场景，例如低压力学习、迁移训练、补救练习或隔离检测；
- 与目标能力相关的用户画像切片；
- 最近相关证据摘要；
- 程序筛选的候选词；
- 难度与提示参数。

真题规律来源、适用范围和版本信息原则上属于 Skill 定义本身的元数据。运行时只需引用对应 Skill 版本，不必每次完整传入。

### 公共输出语义

公共输出至少包括：

- 本轮训练目标；
- 材料或任务规格；
- 生成内容；
- 本任务预期观察的能力与证据；
- Skill 版本与生成依据；
- 需要执行的质量校验要求。

### 客观训练题或隔离检测载荷

客观训练题或隔离检测题的任务载荷应额外包含：

- 标准答案；
- 可回指材料的答案依据；
- 干扰项错误依据；
- 可能暴露的错误类型。

### 低压力学习、拆解和补救载荷

低压力学习、长难句拆解或补救讲解不强制包含标准答案和干扰项依据。

这类任务应输出适合任务类型的：

- 分层提示；
- 讲解结构；
- 练习要求；
- 可观察行为或理解证据。

### 质量校验要求与结果

必须区分：

- Skill 声明的质量校验要求；
- 程序或独立校验环节实际产生的质量校验结果。

不能把模型自称“已检查通过”当作质量校验结果。

## 4 类 CET-6 阅读 Skill

### 1. 词汇语境识别 Skill

目标能力：

- 在阅读语境中识别目标词、熟词僻义、固定搭配和语义倾向；
- 将词义判断绑定到上下文，而不是孤立背诵释义。

适用条件：

- 用户词汇识别弱；
- 用户在上下文中误解词义；
- 用户对目标词频繁点击释义；
- 主线候选池中存在需要复习或迁移的词汇。

输入与输出结构：

- 使用统一公共契约；
- 低压力学习可输出目标词语境、释义提示和短练习；
- 客观训练题需输出标准答案、答案依据和错误选项依据。

可调难度参数：

- 目标词数量；
- 目标词在文章中的显性程度；
- 上下文提示强弱；
- 是否包含熟词僻义；
- 是否包含固定搭配；
- 提示等级。

基于真题规律的训练或题目生成规则：

- 词义判断应依赖上下文线索；
- 题目不能只考词典释义；
- 干扰项可来自常见近义词、脱离语境释义或错误搭配；
- 目标词应自然嵌入 CET-6 风格阅读材料。

答案依据：

- 答案必须能回指到上下文线索、搭配关系或句内外语义关系。

常见错误类型：

- 孤立套用熟悉释义；
- 忽略搭配；
- 忽略上下文限定；
- 混淆近义词；
- 被中文直觉误导。

最小质量校验规则：

- 目标词实际出现在材料中；
- 目标词语境支持标准答案；
- 客观题答案唯一；
- 干扰项与错误类型对应；
- 题目不依赖未提供背景知识。

### 2. 长难句与逻辑关系 Skill

目标能力：

- 理解长句主干、修饰成分、指代关系和句间逻辑；
- 识别转折、让步、因果、递进、对比等关系对答案的影响。

适用条件：

- 用户能认识词但理解句意失败；
- 用户依赖句子提示或拆解；
- 用户在逻辑关系相关题目中出错；
- 阅读材料中包含影响理解的复杂句。

输入与输出结构：

- 使用统一公共契约；
- 拆解任务可输出句子分层、主干、修饰关系和逻辑提示；
- 客观训练题需输出答案依据和干扰项错误依据。

可调难度参数：

- 句长；
- 从句数量；
- 插入结构复杂度；
- 指代距离；
- 逻辑连接词显性程度；
- 提示等级。

基于真题规律的训练或题目生成规则：

- 训练应围绕影响理解和答题的句法或逻辑关系；
- 不做纯语法术语测试；
- 题目应要求用户利用句间关系或句子主干判断含义；
- 干扰项可利用转折前信息、因果倒置、指代误解或范围误判。

答案依据：

- 答案必须能回指到句子结构、逻辑连接、指代对象或上下文关系。

常见错误类型：

- 抓错句子主干；
- 忽略转折或让步；
- 因果关系倒置；
- 指代对象判断错误；
- 把修饰信息当成核心结论。

最小质量校验规则：

- 被训练句子确实包含目标结构或逻辑关系；
- 讲解能区分主干与修饰；
- 客观题答案唯一；
- 答案依据可回指材料；
- 干扰项与常见误解对应。

### 3. 同义替换与定位 Skill

目标能力：

- 在题干、选项和原文之间识别同义替换；
- 根据题干线索定位原文相关信息；
- 区分原文信息与选项改写之间的关系。

适用条件：

- 用户能看懂原文局部但找不到答案位置；
- 用户无法识别题干或选项的同义改写；
- 用户在细节题、推断题或信息定位中失误；
- 第二次自适应短训练任务需要强化真题风格迁移训练。

输入与输出结构：

- 使用统一公共契约；
- 客观训练题需输出标准答案、答案依据、同义替换对应关系和干扰项依据。

可调难度参数：

- 定位跨度；
- 同义替换抽象程度；
- 原文与题干词面重合度；
- 选项迷惑度；
- 题干线索显性程度；
- 提示等级。

基于真题规律的训练或题目生成规则：

- 正确选项不应只是复制原文；
- 题干和答案依据之间应存在可解释的改写关系；
- 干扰项可使用原文词面、相邻信息或局部正确但整体错误的信息；
- 定位线索必须客观存在于材料中。

答案依据：

- 答案必须能回指到原文具体句子或段落；
- 同义替换关系必须可说明。

常见错误类型：

- 只找原词不找同义表达；
- 被原文词面重复的干扰项吸引；
- 定位到相邻但错误的信息；
- 忽略题干限定条件；
- 混淆局部正确与整体正确。

最小质量校验规则：

- 题干存在可定位线索；
- 正确答案与原文有可解释同义替换；
- 干扰项不是无关乱造；
- 答案唯一；
- 答案依据可回指材料。

### 4. 干扰项判断 Skill

目标能力：

- 识别 CET-6 阅读常见干扰项模式；
- 判断选项为什么错，而不是只凭直觉选对；
- 避免过度推断、偷换概念、范围扩大或缩小、无中生有。

适用条件：

- 用户能定位原文但选错；
- 用户经常在两个选项间犹豫；
- 用户容易被看似相关的信息误导；
- 训练目标从理解材料转向提高选项判断质量。

输入与输出结构：

- 使用统一公共契约；
- 客观题必须输出标准答案、答案依据、每个干扰项的错误依据和可能暴露的错误类型。

可调难度参数：

- 干扰项数量；
- 干扰项与原文相关程度；
- 干扰模式复杂度；
- 选项间差异显性程度；
- 是否包含多种干扰模式；
- 提示等级。

基于真题规律的训练或题目生成规则：

- 干扰项应来自真实阅读题常见错误模式；
- 错误选项应有迷惑性，但必须可被材料证据排除；
- 不允许靠常识争议排除；
- 不允许多个选项都可成立。

答案依据：

- 正确答案依据必须回指材料；
- 每个干扰项的错误原因必须能对应材料证据或题干限制。

常见错误类型：

- 过度推断；
- 偷换概念；
- 范围扩大；
- 范围缩小；
- 无中生有；
- 张冠李戴；
- 把部分事实当成题目答案。

最小质量校验规则：

- 正确答案唯一；
- 每个干扰项都有明确错误依据；
- 干扰项属于可标注模式；
- 题目不依赖材料之外的主观判断；
- 答案解释能说明为什么其他选项错。

## 隔离检测中的职责

隔离检测题来自预先保留的未见题集或合法样题风格保留题。

MVP 中，隔离检测发生在第二次自适应短训练任务之后。第二次主线不要求再次完成一套完整学习闭环，也不要求在隔离检测前再次完成完整画像更新。

Agent 不得：

- 在训练时访问隔离题集；
- 根据用户本轮表现临时生成隔离题后用于自证；
- 在检测过程中提供提示；
- 检测中途调整难度；
- 把检测解释提前暴露给用户。

程序负责：

- 控制题集隔离；
- 展示检测题；
- 计时；
- 判分；
- 保存题目版本、用户答案、用时和结果。

隔离检测结果可作为更高置信度证据，但仍需程序按规则校验后才能影响正式画像。

## 副线中的职责

Agent 可以：

- 将主线薄弱词映射到机场场景曝光；
- 根据副线信号建议后续主线验证；
- 解释副线词汇或表达；
- 让副线任务保持低压力和语境化。

Agent 不得：

- 根据副线任务完成情况直接更新正式能力画像；
- 把副线选对航班解释为 CET-6 阅读能力提升；
- 将副线数据作为隔离检测证据；
- 让副线内容泄漏隔离题集。

副线信号只能作为候选曝光或待验证信号。
